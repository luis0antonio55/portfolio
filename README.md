## **Local configuration** 
1. Clone the repo:  
```
git clone https://github.com/luis0antonio55/portfolio
```
2. Install dependencies:
```  
npm install
```
3. Start the development server:
```  
npm run dev
```
for infisical use:
```  
infisical run -- npm run dev
```
